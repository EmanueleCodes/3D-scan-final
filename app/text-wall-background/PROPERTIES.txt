Text Wall - Property Controls

Preview
Toggle animation on/off in the Framer canvas. Turn on to see the animation effect while editing.

Words
Array of words that will be randomly positioned across the text wall. These words appear scattered among random characters, creating a hacker-terminal aesthetic.

Lines
Number of text lines to display. More lines create a denser wall effect.

Chars/Line
Number of characters per line. Controls the width of each text line.

Words/Line
Maximum number of words to place in each line. Words are randomly distributed within the line.

Text Color
Color of all text (both words and random characters).

Background
Background color of the component.

Font Size
Size of the monospace font in pixels.

Line Gap
Vertical spacing between lines in pixels.

Duration
How long the reveal animation takes for each line (in seconds).

Stagger
Delay between each line's animation start (in seconds). Creates a cascading reveal effect.

Loop
When enabled, the animation continuously cycles - revealing, holding, hiding, and regenerating new random layouts.
